# halleinApp
## Setup
### Server
Install [Node.js](https://nodejs.org/en/download/)

Open commandline in halleinApp/
```
$ npm install
 
$ node server.js
```

### Gulp
Install [Node.js](https://nodejs.org/en/download/)

Open commandline in halleinApp/
```
$ npm install --global gulp-cli
```
In commandline or PhpStorm (just press [CTRL]+[SHIFT]+[X]) type
```
$ gulp
```
